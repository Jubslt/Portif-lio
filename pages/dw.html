Desenvolvimento web

O desenvolvimento web é o processo de criar e manter sites e aplicações web. Este campo abrange uma variedade de tarefas e tecnologias que visam garantir que os sites sejam funcionais, acessíveis e esteticamente agradáveis.

O que aprendemos?

Durante as aulas de desenvolvimento web aprendemos como é o processo completo de criação de um site, desde o design da interface até a implementação do código. A área de desenvolvimento web possui três especializações principais:

Front-end: encarregado da parte visual e interativa do site (HTML, CSS e JavaScript)

Back-end: É atribuído a ele a tarefa de fazer o site funcionar e da parte lógica (Python, Java, PHP e C#)

Full-stack: domina os dois tipos de conhecimento Front-end e Back-end.

Desenvolvemos isso em uma das primeias aulas.

Plataforma usada

De primeira aprendemos a mexer no figma para conseguir ter um protótipo de como ficaria o design do site. Para desenvolver o site com o código usamos o Visual Studio Code onde aprendemos alguns fundamentos que são:

HTML (HyperText Markup Language): estrutura o conteúdo no site

CSS (Cascading Style Sheets): Deixa a estrutura do site com estilo

JavaScript: Dá funcionalidade e cria interatividade para o site
