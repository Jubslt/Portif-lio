Linguagem de Programação

A linguagem de programação é um conjunto de regras que define como escrever código fonte para computadores. Esse código pode ser compilado para criar programas ou interpretado diretamente para executar instruções de processamento.

O que é Java?

Java é uma linguagem de programação de alto nível, orientada a objetos e independente de plataforma, usada para desenvolver aplicações em diversos dispositivos, como computadores, servidores e dispositivos móveis. Usamos durante as aulas o Intellij, que é uma ferramenta de desenvolvimento de software.

Estrutura de decisão:

Permite que o programa tome diferentes caminhos de execução com base em condições especificadas.

If/else: permite executar um bloco de código se a condição for verdadeira, e outro bloco de código se a condição for falsa.

Switch: usada para executar um bloco de código entre várias opções possíveis com base no valor de uma expressão.

Estrutura de Repetição:

Permite que um bloco de código seja executado várias vezes, com base em uma condição.

While: continua executando um bloco de código enquanto a condição especificada for verdadeira.

Do/while: é semelhante ao while, mas garante que o bloco de código seja executado pelo menos uma vez.

For: é usado quando o número de iterações é conhecido previamente.

Vetores:

Vetores são chamados de arrays. Eles são estruturas de dados que permitem armazenar múltiplos valores do mesmo tipo sob um único nome. Array é uma classe abstrata, sendo assim possui alguns métodos e atributos, como por exemplo o atributo length.

Matrizes:

Matrizes são essencialmente arrays bidimensionais, sendo que um vetor é unidimensional. Cada elemento principal é, na verdade, um array. Os valores dessa matriz podem ser inicializados diretamente na declaração. Para percorrer os elementos de uma matriz pode utilizar, por exemplo, um sistema de laços "for" aninhados ou 'foreach". Neste caso, temos que tomar cuidado, pois cada interação do "foreach" não retornará apenas o valor, mas sim o vetor junto.

Funções:

Funções são blocos de código que executam uma determinada tarefa e podem ser chamados em qualquer lugar do programa onde são acessíveis.

Funções recursivas:

Funções recursivas executam a chamada delas no próprio bloco de instruções. Um exemplo é o método 'fatorial' que serve para calcular o fatorial de um número inteiro 'n'.
