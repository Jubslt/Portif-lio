Automação Industrial

A automação industrial usa tecnologias para controlar e monitorar processos, aumentando a eficiência, precisão e segurança. O Arduino, uma placa de microcontrolador acessível, facilita a criação de projetos eletrônicos interativos. Uma aplicação comum é o uso de displays LCD para exibir informações em tempo real.

Arduino

O Arduino é uma plataforma de prototipagem eletrônica de código aberto, fácil de usar em hardware e software. Usamos durante as aulas o Tinkercad, uma plataforma online que permite criar e simular circuitos eletrônicos de forma prática, visualizando o funcionamento dos circuitos sem a necessidade de hardware físico.

Arduino com LCD

Utilizamos um LCD com o Arduino para exibir informações como leituras de sensores, status do sistema e mensagens de erro diretamente no local de operação. LCDs são uma tecnologia de exibição encontrada em dispositivos eletrônicos como smartphones e monitores, usados para mostrar imagens e texto com alta qualidade e baixo consumo de energia.

Linguagem Usada:

Usamos a linguagem C++ nas atividades, pois é a principal linguagem para programar no Tinkercad e no Arduino.

Algumas características específicas da linguagem C++:
Função obrigatória:
  • setup(): Usada para iniciar o programa, sendo chamada uma vez.
  • loop(): Usada para executar o código repetidamente.
Manipulação de pinos:
  • pinMode(pin, mode): Configura o pino como entrada ou saída.
  • digitalWrite(pin, value): Escreve um valor no pino digital.
  • digitalRead(pin): Lê o valor do pino digital.
  • analogRead(pin): Lê o valor do pino analógico.
  • analogWrite(pin, value): Escreve um valor analógico em um pino digital.
Funções de Temporização:
  • delay(ms): Pausa o programa por uma quantidade de milissegundos.
  • millis(): Retorna o tempo em milissegundos desde que o programa começou a ser executado.
