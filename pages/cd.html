Ciência de Dados

Ciência de Dados é um campo interdisciplinar que usa métodos científicos, algoritmos e sistemas para extrair conhecimento e insights de dados estruturados e não estruturados, combinando estatística, matemática, computação e conhecimento do domínio.

População e amostra

  • A estatística trabalha com dados, que podem ser obtidos a partir de uma população ou de uma amostra, definida como:
  • População: conjunto de elementos com uma característica comum que os define, podendo ser animados ou inanimados.
  • Amostra: Subconjunto representativo da população, menor em dimensão, selecionado de várias maneiras. A estatística inferencial trata da metodologia de amostragem.
  • Elementos: Unidades observadas no estudo (Medronho, 2003).
  • Variável: Característica medida, observada ou contada nos elementos, que pode variar.

Amostragem e seus tipos

Amostragem é o processo de selecionar um subconjunto representativo de uma população maior para fazer inferências sobre toda a população. Este método é usado para economizar tempo e recursos enquanto ainda se obtêm resultados confiáveis facilitando a coleta e análise de dados.

Tipos Comuns de Amostragem:
  • Amostragem Aleatória Simples: Todos os elementos da população têm a mesma chance de serem escolhidos.
  • Amostragem Sistemática: Seleciona elementos a intervalos regulares de uma lista ordenada.
  • Amostragem Estratificada: Divide a população em subgrupos homogêneos (estratos) e seleciona amostras aleatórias de cada estrato.

Escalas de Medição

São fundamentais na ciência de dados, pois determinam como os dados são coletados, analisados e interpretados. Existem quatro escalas principais de medição:

  • Escala Nominal:
    Classifica os dados em categorias distintas sem uma ordem específica.
  • Escala Ordinal:
    Classifica os dados em categorias que possuem uma ordem ou hierarquia.
  • Escala Intervalar:
    Mede a diferença entre valores, com intervalos iguais entre pontos, mas sem um ponto zero absoluto.
  • Escala de Razão:
    Similar à escala intervalar, mas com um ponto zero absoluto, permitindo a comparação de magnitudes.

Tendências centrais

As medidas de tendência central são estatísticas que descrevem o centro de um conjunto de dados.

  • Média:
    A média aritmética é a soma de todos os valores dividida pelo número de valores.
  • Média Aritmética Ponderada:
    A média aritmética ponderada é uma média na qual cada valor do conjunto de dados é multiplicado por um peso que representa sua importância relativa.
  • Mediana:
    O valor que separa a metade superior e a metade inferior de um conjunto de dados.
  • Moda:
    O valor que ocorre com maior frequência em um conjunto de dados.
  • Bimodal:
    Refere-se a um conjunto de dados que possui duas modas distintas, ou seja, dois valores que ocorrem com a mesma frequência máxima.
  • Polimodal:
    Indica um conjunto de dados que possui mais de duas modas distintas, ou seja, vários valores que ocorrem com a mesma frequência máxima.
  • Amodal:
    Uma distribuição de dados que não possui uma moda clara. Todos os valores ocorrem com a mesma frequência, ou as frequências são uniformemente distribuídas.
  • Percentil:
    Percentil é uma medida que indica o valor abaixo do qual uma certa porcentagem de observações cai.
  • Quartil:
    Quartis são valores que dividem um conjunto de dados ordenado em quatro partes iguais.

Medidas de dispersão

As medidas de dispersão são usadas para descrever a variabilidade ou a dispersão dos dados em um conjunto de dados. Elas complementam as medidas de tendência central, fornecendo informações sobre como os dados se espalham em torno do centro.

Amplitude:

A diferença entre o valor máximo e o valor mínimo em um conjunto de dados.

Variância:

A variância é o quadrado do desvio padrão e representa a média dos quadrados dos desvios da média.

Desvio Padrão:

Mede o quanto os valores de um conjunto de dados se desviam, em média, da média do conjunto de dados.

Distribuição Probabilística Binomial

A distribuição probabilística binomial é uma distribuição de probabilidade discreta que descreve o número de sucessos em uma sequência de tentativas independentes de um experimento binário. Cada tentativa tem apenas dois resultados possíveis: sucesso ou fracasso.
